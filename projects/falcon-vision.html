Falcon Vision

Abstract

The problem statement was "Creating an affordable solution through camera feed for number plates of vehicles for the detection, identification, and monitoring of vehicles in different scenarios".

Initially, we thought about implementing it in our college. This will remove the hassle of manually entering Car details everytime we enter the campus. Our solution also provides a robust traffic flow management and monitoring system using web app and android app.

Outcome

A central database was created for pushing all the data acquired from the previously described pipeline for ensuring scalability to multiple nodes of operation within a service campus. This also provided real-time data visualizations for an end-user through Android and Web based applications designed for site users and site Admin respectively.

The project is set up for testing in a real world environment to get feedback from potential customers like closed residential areas, industrial parks, college campuses and parking spaces. This will be followed by improving the proposed system based on the feedback received.
